Para ejecutar este proyecto

1 Clonar el repositorio https://github.com/Apocalypsh9t/Nuevas_tecnologias.git

2 Crear el entorno: python -m venv venv

3 Activar el entorno (Windows): .\venv\Scripts\activate

4 Activar el entorno (Mac/Linux): source venv/bin/activate

5 Instalar Dependencias
Una vez activado el entorno, deben instalar las librerías listadas en el archivo de requerimientos:
pip install -r requirements.txt

6 Ejecución del Proyecto
Para probar la lógica base y el menú interactivo, deben ejecutar el script principal:
python usuarios.py
